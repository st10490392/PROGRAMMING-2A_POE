# Cybersecurity Chatbot (POE Part 1)

## Description
This is a console-based chatbot built in C# that educates users about cybersecurity.

## Features
- Voice greeting (.wav)
- ASCII art display
- User name interaction
- Cybersecurity responses
- Input validation

## How to Run
dotnet run
can even run on vscode
